## Description

Ce projet consiste à réaliser un Curriculum Vitae (CV) interactif sous forme de page web en utilisant HTML, CSS et JavaScript.

Le CV présente les informations personnelles, le parcours académique, les compétences, les projets réalisés, les centres d'intérêt ainsi qu'un formulaire de contact.

L'application est responsive et s'adapte aux écrans d'ordinateur, de tablette et de téléphone.

## Fonctionnalités

- Navigation interne avec défilement fluide
- Mode sombre / clair avec sauvegarde dans le localStorage
- Menu hamburger pour les appareils mobiles
- Barres de progression animées pour les compétences
- Formulaire de contact avec validation JavaScript
- Bouton de retour en haut de page
- Téléchargement du CV en PDF 
- Design responsive

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- Font Awesome

## Installation et exécution

1. Télécharger ou cloner le projet.
2. Ouvrir le dossier du projet.
3. Lancer le fichier "index.html" dans un navigateur web (Google Chrome, Mozilla Firefox, Microsoft Edge, etc.).
4. Utiliser le menu de navigation pour parcourir les différentes sections du CV.
